Catatan Penting: Bot ini hanya mendukung model AI dari Gemini saja saat ini.

Sebelum menjalankan program, anda wajib mengatur konfigurasinya dulu agar bot bisa berjalan:

1. Masuk ke folder CypherBot ➔ backend.

2. Cari file .env.example, lalu ubah namanya menjadi .env.

3. Buka file .env tersebut, kemudian isi bagian berikut dengan API Key milikmu:
   GEMINI_API_KEY=Masukkan_API_Key_Gemini_Kamu_Di_Sini
   (API Key bisa didapatkan secara gratis melalui Google AI Studio).

ENG
Important Note: This bot only supports Gemini AI models at the moment.

Before running the program, you are required to set up the configuration first so the bot can run properly:

1. Go to the CypherBot ➔ backend folder.

2. Find the .env.example file and rename it to .env.

3. Open the .env file, then fill in the following section with your API Key:
GEMINI_API_KEY=Paste_Your_Gemini_API_Key_Here
(You can get the API Key for free via Google AI Studio)
